# Express ve Nodejs HTTP Farkları

1) Express kod yazımı olarak çok daha sadedir.

2) Express belirli rotalara atılan istekleri tek başına yönetebilir.

3) Zincirleme kod yazım metodu ile aynı kodu daha basit yazar.

4) Modüler yapılıdır, fonksiyonları ayrı yerde yazıp istediğiniz yerde çağırırız.
Yani rota mantığımız ayrı dosyada, isteklere gönderilecek cevaplar ayrı dosyada tutulabilir.

5) Express'in "middleware" adını verdiğimiz aracı fonksiyonları mevcuttur. 
Bu aracılar asıl cevabı biz göndermeden önce veriyi bizim işimize yarayacak şekilde günceller.
