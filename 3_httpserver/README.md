# Nodejs projesi oluşturma

> npm init

Package.json oluşturma sihirbazını açar ve proje oluşturmak için bazı sorular sorar.

> npm init -y

Aynı sihirbazı açar fakat bütün sorulara ENTER basılmış gibi hızlıca oluşturur.

# Node.js kullanarak bir HTTP Sunucusu (server) Oluşturmak

Normalde JS'in sadece tarayıcı ortamlarında çalışabildiğini hatırlamak gerekir.

Bu sebepten oluşturduğumuz JavaScript dosyalarını Node.js kullanarak ayağa kaldırmak şarttır.
(node komutunu kullanarak ÖRN. "node server.js" )
